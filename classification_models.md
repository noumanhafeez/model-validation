# Classification Models (Simple Explanation)

## What is a Classification Model?
A classification model is a machine learning model used when the **output is a category instead of a number**.

## Examples of Classification Problems
Examples where classification models are used:

- Predict **hair color of a newborn** (black, brown, blonde).
- Predict **winner of a basketball game** (team A or team B).
- Predict **genre of the next song** (rock, pop, jazz).

In these cases, the output is **categorical**, not a number.

# Example 1: Email Spam Detection

## Problem
We want to predict whether an email is **Spam** or **Not Spam**.

## Input Features
The model might use features such as:

- Number of suspicious words
- Email length
- Presence of links
- Sender email address

## Output Classes

```text
Spam
Not Spam
